# مسیریابی دی‌ان‌اس هوشمند (dns-smart-routing) برای OpenWRT

یک سیستم سوئیچ خودکار دی‌ان‌اس (DNS Failover) برای روترهای OpenWRT جهت بهبود پایداری اتصال دی‌ان‌اس در شبکه‌های دارای اختلال و کندی (مانند ایران).

## توضیحات مهم و سلب مسئولیت (تضمین ایمنی)
- ❌ **این ابزار به هیچ عنوان فیلترشکن، پروکسی یا VPN نیست.**
- ❌ این ابزار هیچ‌گونه قابلیت دور زدن فیلترینگ ندارد.
- ❌ این ابزار تغییری در فایروال یا جداول مسیریابی سیستم ایجاد نمی‌کند.
- این پروژه **صرفاً** پایداری و سلامت پاسخ‌دهی سرورهای دی‌ان‌اس را بررسی کرده و در صورت بروز قطعی، دی‌ان‌اس روتر را به سرور محلی امن سوییچ می‌کند.

## نحوه کارکرد (مدل ۲ وضعیتی)
- **وضعیت NORMAL (عادی)**: استفاده از دی‌ان‌اس‌های پیش‌فرض شبکه WAN (بدون تغییر).
- **وضعیت FAILOVER (پشتیبان)**: استفاده از دی‌ان‌اس محلی امن بر روی آدرس `127.0.0.1#15353`.
- **شروط انتقال**: بروز ۴ خطای متوالی (تاخیر بالای ۳۰۰ میلی‌ثانیه یا عدم پاسخ) اولویت را به وضعیت `FAILOVER` می‌برد. ثبت ۸ موفقیت متوالی، روتر را به وضعیت `NORMAL` باز می‌گرداند.

## نحوه نصب

```bash
chmod +x install.sh
./install.sh
```
