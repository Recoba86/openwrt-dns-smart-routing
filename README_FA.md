# مسیریابی دی‌ان‌اس هوشمند (dns-smart-routing v2) برای OpenWRT

یک سرویس سبک و پایدار برای بررسی و حل اختلالات دی‌ان‌اس (DNS Timeout, Poisoning, Hijack) در روترهای OpenWRT (بهینه‌سازی شده برای شبکه‌های دارای محدودیت و اختلال مانند ایران).

## مشکل کاربران در ایران
در شبکه‌های دارای محدودیت و فیلترینگ:
- دسترسی به سرورهای دی‌ان‌اس خارجی با قطعی یا پاسخ‌های نامعتبر مواجه می‌شود.
- پدیده مسموم‌سازی دی‌ان‌اس (DNS Poisoning) مانع از حل صحیح آدرس‌ها می‌گردد.
- زمان تاخیر پاسخ‌دهی دی‌ان‌اس بسیار طولانی یا منقطع است.

## توضیح ساده سیستم (مدل ۳ وضعیتی)
- **وضعیت CLEAN (پاک)**: دی‌ان‌اس عادی روتر بدون تغییرات.
- **وضعیت DEGRADED (مختل)**: مسیریابی ترکیبی. کوئری‌ها به صورت موازی به دی‌ان‌اس‌های عمومی و محلی فرستاده می‌شوند.
- **وضعیت BROKEN (خراب)**: مسیریابی فقط از طریق دی‌ان‌اس محلی امن (`127.0.0.1#15353`).

## تاکید مهم
- ❌ **این ابزار فیلترشکن، پروکسی یا VPN نیست.**
- ❌ تغییری در مسیرهای شبکه یا فایروال ایجاد نمی‌کند و فقط سرور حل دی‌ان‌اس dnsmasq را بازنشانی می‌نماید.
